# 案件概要

株式会社テクノ精工は金属部品の中小製造業(従業員 80 名)で、現在 Excel と紙帳票で在庫を管理している。
品番数が約 3,000 点に増加し、月次棚卸しに丸 2 日かかるほか、欠品による生産停止が月平均 2 回発生している。
Web ベースの在庫管理システムへ刷新し、リアルタイムな在庫可視化・自動発注アラート・バーコードスキャン対応を実現したい。
将来的には販売管理システム(既存 X 社製)との API 連携も視野に入れている。
