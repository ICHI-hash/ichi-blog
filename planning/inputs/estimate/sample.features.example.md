# 機能リスト: 簡易タスク管理ツール

### F-001: ユーザー登録 / ログイン
- 概要: メールアドレスとパスワードで新規登録およびログイン。JWT トークン認証。
- 入力: email, password
- 出力: 認証トークン(JWT)
- 受け入れ基準: 既存メールでの重複登録不可、ログイン失敗 5 回でアカウントロック、パスワードは bcrypt でハッシュ保存

### F-002: タスク CRUD
- 概要: タスクの作成・閲覧・更新・削除。タイトル・期限・ステータス・担当者を管理。
- 入力: title, description, due_date, status(todo/in_progress/done), assignee_id
- 出力: タスクオブジェクト
- 受け入れ基準: 自分が作成したタスクのみ編集・削除可能、閲覧はチームメンバー全員可

### F-003: タスクへのラベル・優先度設定
- 概要: 各タスクに複数ラベルおよび優先度(高 / 中 / 低)を付与する。
- 入力: label_ids[], priority(high/medium/low)
- 出力: 更新されたタスクオブジェクト
- 受け入れ基準: ラベルは最大 10 個まで付与可能、優先度は必須項目

### F-004: チームメンバーへのタスク割り当て
- 概要: タスクを同一チームの別メンバーにアサインする。
- 入力: task_id, assignee_user_id
- 出力: 割り当て済みタスクオブジェクト、担当者へのメール通知
- 受け入れ基準: 同一チームのメンバーにのみ割り当て可能、割り当て時にメール通知を送信

### F-005: 期限アラート通知
- 概要: タスクの締め切り 1 日前と当日に担当者へメール通知を送信する。
- 入力: タスク期限日、担当者メールアドレス
- 出力: 通知メール
- 受け入れ基準: バックグラウンドジョブで日次実行、完了済みタスクには通知しない
