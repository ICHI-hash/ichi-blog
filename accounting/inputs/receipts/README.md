# accounting/inputs/receipts/

経費の領収書・レシートを月別に配置するディレクトリです。

## フォルダ構造

```
receipts/
└── YYYY-MM/          ← yyyymm 形式(STEP 7 の引数と完全一致させること)
    ├── 20260510-amazon-3300.pdf
    ├── 20260515-cafe-1200.jpg
    └── ...
```

## ファイル名規約

**推奨**: `<YYYYMMDD>-<取引先>-<金額>.<拡張子>`

例: `20260510-amazon-3300.pdf`、`20260515-cafe-1200.jpg`

必須ではありませんが、取引先と金額をファイル名に含めると
税理士の確認が容易になります。

## 注意

- PDF・JPG・PNG などの画像ファイルを直接配置してください
- このディレクトリは `.gitignore` で追跡対象外です
- `tax-package` コマンドが `06-receipts/` にコピーします
